# Jumble provides on-demand labor solutions by matching employers with temporary workers in an area

# Features
## Jumble users can use their profile to either post jobs that they want done or search through existing job postings for work. 
## Customize user profiles with self-identifying information, such as skills or profile image.
## Sign-In and Sign-Up using Firebase Authentication
## Users can search, filter, and browse for jobs
## Interested workers can indicate interest on job postings that employers can respond to.
